Module Dungeon

module Dungeon: sig .. end

Representation of the dungeon.

This module represents the data needed to maintain and render the dungeon, including a data about each tile. Includes methods for obtaining information about the dungeon.

type tile = {
   material : string;
   is_wall : bool;
   item : Item.t option;
   npc : Npc.t option * int * int;
}

an abstract type representing a tile

type cell = {
   tile : tile;
   x : int;
   y : int;
}

an abstract type representing a dungeon cell

   dimensions : int * int;
   bound : int;
   next : int;
   prev : int;
   magic_numbers : Magic_numbers.t;
   time : int;
}

an abstract type representing a dungeon

val is_wall : t -> int * int -> bool

is_wall dungeon (x, y) is true iff the cell at coordinate (x, y) is a wall in dungeon

val get_item : t -> int * int -> Item.t option

get_item dungeon (x,y) returns the Item.t option associated with tile (x,y), and then removes the item from that tile

val drop_item : t -> int * int -> Item.t option -> unit

drop_item dungeon (x,y) item places (item) on the tile (x,y)

val get_start : t -> int * int

get_start dungeon is the coordinates of the starting tile in dungeon

val get_prev : t -> int

get_prev dungeon is the id of the dungeon that precedes dungeon

val get_next : t -> int

get_next dungeon is the id of the dungeon that follows dungeon

val get_exit : t -> int * int

val get_dimensions : t -> int * int

val get_time : t -> int
